svn 要回復已 commit 的修改很簡單。直接 merge backward diff 就可以了。

```
svn merge -r n:n-1 ^/trunk
```

參考 [stackoverflow: How do I return to an older version of our code in Subversion?][1] 與 [SVN redbook][2]
