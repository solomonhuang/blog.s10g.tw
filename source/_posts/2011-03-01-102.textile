測試一下 gzip, bzip 與 xz 的表現。

被壓縮檔是 ubuntu-10.10-desktop-i386.iso，大小為694MB(726827008 bytes)。

第一次測試，單純測試時間，指定參數全部相同為‘-c ubuntu-10.10-desktop-i386.iso &gt; /dev/null’。
{% codeblock %}======== GZIP ========
real	0m44.138s
user	0m42.300s
sys	0m0.576s

======== BZIP2 ========
real	4m55.145s
user	4m53.074s
sys	0m0.750s

======== XZ ========
real	7m53.939s
user	7m51.902s
sys	0m1.436s{% endcodeblock %}
第二次測試，主要紀錄壓縮檔大小，指定參數均為‘-c ubuntu-10.10-desktop-i386.iso &gt; u1010.iso.gz’
{% codeblock %}======== GZIP ========
real	0m44.551s
user	0m42.279s
sys	0m1.303s
684M(716987381 bytes) u1010.iso.gz

======== BZIP2 ========
real	4m52.963s
user	4m50.558s
sys	0m1.704s

689M(721999239 bytes) u1010.iso.bz2

======== XZ ========
real	8m3.213s
user	7m57.394s
sys	0m2.654s
679M(711776308 bytes) u1010.iso.xz{% endcodeblock %}
測試的script

第一次：
{% codeblock %}#!/bin/sh
echo ======== GZIP ========
time gzip -c ubuntu-10.10-desktop-i386.iso &gt; /dev/null
echo ======== BZIP2 ========
time bzip2 -c ubuntu-10.10-desktop-i386.iso &gt; /dev/null
echo ======== XZ ========
time xz -c ubuntu-10.10-desktop-i386.iso &gt; /dev/null{% endcodeblock %}
第二次：
{% codeblock %}#!/bin/sh
echo ======== GZIP ========
time gzip -c ubuntu-10.10-desktop-i386.iso &gt; u1010.iso.gz
ls -sh u1010.iso.gz
echo ======== BZIP2 ========
time bzip2 -c ubuntu-10.10-desktop-i386.iso &gt; u1010.iso.bz2
ls -sh u1010.iso.bz2
echo ======== XZ ========
time xz -c ubuntu-10.10-desktop-i386.iso &gt; u1010.iso.xz
ls -sh u1010.iso.xz{% endcodeblock %}
